## SEP 状态定义

| 状态                                                | 定义                                               |
| ----------------------------------------------------- | -------------------------------------------------------- |
| <Badge color="gray" shape="pill">草稿</Badge>        | 带有发起人的 SEP 提案，正在进行非正式审查  |
| <Badge color="yellow" shape="pill">审查中</Badge>  | SEP 提案已准备好供核心维护者正式审查 |
| <Badge color="blue" shape="pill">已接受</Badge>     | SEP 已接受，等待参考实现          |
| <Badge color="green" shape="pill">最终版</Badge>       | SEP 已定稿，参考实现完成     |
| <Badge color="red" shape="pill">已拒绝</Badge>      | SEP 被核心维护者拒绝                         |
| <Badge color="red" shape="pill">已撤回</Badge>     | SEP 被作者撤回                              |
| <Badge color="purple" shape="pill">已取代</Badge> | SEP 被新的 SEP 取代                              |
| <Badge color="orange" shape="pill">休眠</Badge>    | 没有发起人的 SEP，6 个月后关闭             |
