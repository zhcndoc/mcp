## 概述

这些条款（“条款”）管辖您对官方 MCP 注册表（“注册表”）的访问和使用（托管在 [https://registry.modelcontextprotocol.io/](https://registry.modelcontextprotocol.io/) 或后续位置的服务），包括提交或发布 MCP 服务器、引用 MCP 服务器或有关此类服务器和/或其开发者的数据（“注册表数据”），以及相关行为。注册表旨在成为一个集中式仓库，存放社区成员开发的 MCP 服务器，以便于 AI 应用程序轻松访问。

这些条款受加利福尼亚州法律管辖。

## 适用于所有用户

1. 无保证。注册表按“原样”提供，没有任何形式的保证。这意味着我们不保证注册表、注册表中包含的服务器或注册表数据的准确性、完整性、安全性、耐用性或可用性。简而言之，我们也不对任何 MCP 服务器或注册表数据负责，我们强烈建议您在决定是否使用每个 MCP 服务器之前，评估其及其对您预期用例的适用性。

2. 访问和使用要求。要访问或使用注册表，您必须：
   1. 年满 18 岁。
   2. 仅以符合美国或其他国家（包括您居住的国家或您从中访问和使用注册表的国家）适用法律的方式使用注册表、注册表中的 MCP 服务器和注册表数据，且不被此类法律禁止访问或使用注册表。您将遵守所有适用的法律、法规和第三方权利（包括但不限于有关数据或软件进出口、隐私、知识产权和当地法律的法律）。您不会使用注册表、MCP 服务器或注册数据来鼓励或促进非法活动或侵犯第三方权利或服务条款。
   3. 通过注册表维护者批准的方法登录，这可能涉及使用第三方拥有的应用程序或其他软件。

3. 实体使用。如果您代表实体访问或使用注册表，您声明并保证您有权使该实体受这些条款的约束。通过接受这些条款，您是代表该实体这样做的（这些条款中对“您”的所有引用均指该实体）。

4. 账户信息。为了访问或使用注册表，您可能需要提供某些信息（例如身份识别或联系方式），作为注册过程的一部分或与您的访问或使用注册表或其中的 MCP 服务器相关联。您提供的任何信息必须准确且最新，您同意及时通知我们任何更新。您了解我们对您使用注册表的情况可能会进行监控，以确保质量并验证您对这些条款的遵守情况。

5. 反馈。您没有义务提供反馈或建议。如果您提供有关注册表或模型上下文协议的反馈或建议，那么我们（以及我们允许的人）可以使用此类信息，而无需对您承担任何义务。

6. 品牌标识。仅在明确指代注册表且不暗示隶属关系、背书或赞助的情况下使用“官方 MCP 注册表”一词。例如，您可以允许说"Acme Inc. 通过自动从官方 MCP 注册表拉取数据来保持其数据最新”或“此数据来自官方 MCP 注册表”，但不能说“这是官方 MCP 注册表的网站”、“我们是查看官方 MCP 注册表数据的首选目的地”或“我们与官方 MCP 注册表合作提供此数据”。

7. 修改。我们可能会修改条款或其任何部分，例如，以反映法律变更或模型上下文协议的变更。我们将在此网站或后续位置发布条款修改的通知。如果您不同意修改后的条款，您应停止访问和/或使用注册表。您继续访问和/或使用注册表即构成您对任何修改后条款的接受。

8. 附加条款。根据您的预期用例，您还必须遵守以下适用条款。

## 适用于 MCP 开发者

9. 禁止事项。通过访问和使用注册表，包括提交 MCP 服务器和/或注册表数据，您同意不：
   1. 分享恶意或有害内容，例如恶意软件，即使是出于善意或研究目的，或执行任何旨在引入任何病毒、蠕虫、缺陷、特洛伊木马、恶意软件或任何破坏性项目的行为；
   2. 诽谤、虐待、骚扰、跟踪或威胁他人；
   3. 干扰或破坏注册表或任何关联服务器或网络；
   4. 提交旨在混淆或误导他人的数据，包括但不限于通过垃圾邮件、发布离题的营销内容、发布以虚假暗示与第三方隶属关系或背书的方式发布 MCP 服务器，或以不同名称重复发布相同或类似的 MCP 服务器；
   5. 推广或促进非法在线赌博或破坏性的商业信息或广告；
   6. 将注册表用于任何注册表的使用或故障可能导致死亡、人身伤害或环境损害的活动；
   7. 使用注册表处理或存储任何受美国国务院维护的国际武器贸易条例管辖的数据。

10. 许可。您同意您提交的关于 MCP 服务器的元数据（例如，架构名称和描述、URL、标识符）和其他注册表数据旨在公开，并将根据 [CC0 1.0 通用](https://creativecommons.org/publicdomain/zero/1.0/) 致力于公共领域。通过提交此类数据，您同意您有权进行此奉献（即，您拥有这些提交内容的版权或已获得版权所有者的许可这样做）并打算这样做。您了解此奉献是永久的、不可撤销的且全球有效的，您在法律允许的最大范围内放弃您对贡献可能拥有的任何精神权利。此奉献仅适用于注册表数据，不适用于您可能指向的第三方注册表中的包。

11. 隐私和公开。您了解您发布的任何 MCP 服务器元数据可能会被公开。这包括个人数据，例如您的 GitHub 用户名、域名或服务器描述中的详细信息。此外，您了解其他人可能会处理包含在您 MCP 服务器元数据中的个人信息。例如，子注册表可能会通过添加您的 GitHub 仓库有多少星来丰富此数据，或对您的代码执行自动安全扫描。通过发布服务器，您同意其他人可以参与此类处理，并且您放弃在某些司法管辖区可能拥有的访问、纠正、删除、限制或反对此类处理的权利。
