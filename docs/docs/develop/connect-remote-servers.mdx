远程 MCP 服务器将 AI 应用程序的功能扩展到本地环境之外，提供对互联网托管工具、服务和数据源的访问。通过连接到远程 MCP 服务器，您将 AI 助手从有用的工具转变为知情的团队成员，能够处理复杂的多步骤项目，并实时访问外部资源。

许多客户端现在支持远程 MCP 服务器，实现了广泛的集成可能性。本指南演示了如何使用 [Claude](https://claude.ai/) 作为示例连接到远程 MCP 服务器，它是 [支持 MCP 的众多客户端之一](/clients)。虽然我们通过自定义连接器关注 Claude 的实现，但这些概念广泛适用于其他兼容 MCP 的客户端。

## 了解远程 MCP 服务器

远程 MCP 服务器的功能类似于本地 MCP 服务器，但它们托管在互联网上而不是您的本地机器上。它们暴露工具、提示和资源，Claude 可以使用它们代表您执行任务。这些服务器可以与各种服务集成，例如项目管理工具、文档系统、代码仓库以及任何其他启用 API 的服务。

远程 MCP 服务器的关键优势在于其可访问性。与需要在每台设备上安装和配置的本地服务器不同，远程服务器可通过任何具有互联网连接的 MCP 客户端使用。这使得它们非常适合基于 Web 的 AI 应用程序、强调易用性的集成以及需要服务器端处理或身份验证的服务。

## 什么是自定义连接器？

自定义连接器充当 Claude 和远程 MCP 服务器之间的桥梁。它们允许您将 Claude 直接连接到对您工作流最重要的工具和数据源，使 Claude 能够在您喜欢的软件中运行，并从外部工具的完整上下文中汲取见解。

使用自定义连接器，您可以：

- [连接 Claude 到第三方开发人员提供的现有远程 MCP 服务器](https://support.anthropic.com/en/articles/11175166-getting-started-with-custom-connectors-using-remote-mcp)
- [构建您自己的远程 MCP 服务器以连接任何工具](https://support.anthropic.com/en/articles/11503834-building-custom-connectors-via-remote-mcp-servers)

## 连接到远程 MCP 服务器

将 Claude 连接到远程 MCP 服务器的过程涉及通过 [Claude 界面](https://claude.ai/) 添加自定义连接器。这在 Claude 和您选择的远程服务器之间建立了安全连接。

<Steps>
<Step title="导航到连接器设置">
在浏览器中打开 Claude 并导航到设置页面。您可以通过点击个人资料图标并从下拉菜单中选择“设置”来访问此页面。进入设置后，找到并点击侧边栏中的“连接器”部分。

这将显示您当前配置的连接器并提供添加新连接器的选项。

</Step>

<Step title="添加自定义连接器">
在连接器部分，滚动到底部，您将找到“添加自定义连接器”按钮。点击此按钮开始连接过程。

<Frame>
  <img
    src="/images/quickstart-remote/1-add-connector.png"
    alt="Claude 设置中的添加自定义连接器按钮"
  />
</Frame>

将出现一个对话框，提示您输入远程 MCP 服务器 URL。此 URL 应由服务器开发人员或管理员提供。输入完整的 URL，确保它包含正确的协议 (https://) 和任何必要的路径组件。

<Frame>
  <img
    src="/images/quickstart-remote/2-connect.png"
    alt="用于输入远程 MCP 服务器 URL 的对话框"
  />
</Frame>

输入 URL 后，点击“添加”以继续连接。

</Step>

<Step title="完成身份验证">
大多数远程 MCP 服务器需要身份验证以确保对其资源的安全访问。身份验证过程因服务器实现而异，但通常涉及 OAuth、API 密钥或用户名/密码组合。

<Frame>
  <img
    src="/images/quickstart-remote/3-auth.png"
    alt="远程 MCP 服务器的身份验证屏幕"
  />
</Frame>

遵循服务器提供的身份验证提示。这可能会将您重定向到第三方身份验证提供商或在 Claude 中显示表单。身份验证完成后，Claude 将与远程服务器建立安全连接。

</Step>

<Step title="访问资源和提示">
成功连接后，远程服务器的资源和提示将在您的 Claude 对话中可用。您可以通过点击消息输入区域中的回形针图标来访问这些内容，这将打开附件菜单。

<Frame>
  <img
    src="/images/quickstart-remote/4-select-resources-menu.png"
    alt="显示可用资源的附件菜单"
  />
</Frame>

菜单显示来自已连接服务器的所有可用资源和提示。选择您要包含在对话中的项目。这些资源为 Claude 提供来自外部工具的上下文和信息。

<Frame>
  <img src="/images/quickstart-remote/5-select-prompts-resources.png" alt="从菜单中选择特定资源和提示" />
</Frame>
</Step>

<Step title="配置工具权限">
远程 MCP 服务器通常公开具有不同功能的多个工具。您可以通过在连接器设置中配置权限来控制允许 Claude 使用哪些工具。这确保 Claude 仅执行您明确授权的操作。

<Frame>
  <img
    src="/images/quickstart-remote/6-configure-tools.png"
    alt="工具权限配置界面"
  />
</Frame>

导航回连接器设置并点击已连接的服务器。在这里，您可以根据需要启用或禁用特定工具、设置使用限制以及配置其他安全参数。

</Step>
</Steps>

## 使用远程 MCP 服务器的最佳实践

使用远程 MCP 服务器时，请考虑这些建议以确保安全高效的体验：

**安全考虑**：连接前始终验证远程 MCP 服务器的真实性。仅连接来自可信来源的服务器，并审查身份验证期间请求的权限。谨慎授予对敏感数据或系统的访问权限。

**管理多个连接器**：您可以同时连接到多个远程 MCP 服务器。按目的或项目组织连接器以保持清晰。定期审查并删除不再使用的连接器，以保持工作区整洁和安全。
