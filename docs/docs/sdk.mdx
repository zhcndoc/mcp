## 快速开始

每个 SDK 提供相同的功能，但遵循其语言的惯用法和最佳实践。所有 SDK 均支持：

- 创建暴露工具、资源和提示词的 MCP 服务器
- 构建可连接任何 MCP 服务器的 MCP 客户端
- 本地和远程传输协议
- 具有类型安全性的协议合规性

访问您所选语言的 SDK 页面以查找安装说明、文档和示例。
