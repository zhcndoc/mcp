---
title: 概述
---

服务器通过 MCP 为语言模型添加上下文提供了基础构建块。这些原语使客户端、服务器和语言模型之间能够实现丰富的交互：

- **提示词**：预定义的模板或指令，用于指导语言模型交互
- **资源**：结构化数据或内容，为模型提供额外上下文
- **工具**：可执行函数，允许模型执行操作或检索信息

每个原语可以在以下控制层次结构中进行总结：

| 原语 | 控制方 | 描述 | 示例 |
| --------- | ---------------------- | -------------------------------------------------- | ------------------------------- |
| 提示词 | 用户控制 | 由用户选择触发的交互式模板 | 斜杠命令、菜单选项 |
| 资源 | 应用程序控制 | 由客户端附加和管理的上下文数据 | 文件内容、git 历史 |
| 工具 | 模型控制 | 暴露给大语言模型以执行操作的函数 | API POST 请求、文件写入 |
